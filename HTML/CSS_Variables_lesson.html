    LEARNING CSS VARIABLES

  • CSS Variables

    ကျွန်တော်တို့တွေက CSS တွေကိုထပ်ခါထပ်ခါအများကြီးရေးမိတဲ့အခါမှာ ကျွန်တော်တို့ web page မှာအလုပ်လုပ်တာနှေးကွေးသွားပါလိမ့်မယ်။ကျွန်တော်တို့တွေရဲ့ style sheet ထဲမှာ value တွေကိုထည့်ဖို့အတွက် variables တွေကိုကြေငြာပြီးအသုံးပြုခဲ့ကြပြီးဖြစ်ပါတယ်။ Variables တွေကို CSS ထဲမှာတိုက်ရိုက်သုံးမယ်ဆိုရင်တော့ ကောင်းမွန်တဲ့ processing tools တွေမလိုအပ်ပါဘူး။ Variables တွေက programming ထဲမှာအရေးအပါဆုံး concept တစ်ခုဖြစ်ပါတယ်။ကျွန်တော်တို့ ရဲ့ stylesheet ထဲမှာပြန်ပြီးအသုံးပြုနိုင်ပြီး CSS variables တွေက values တွေကိုရည်ညွန်းဖို့အတွက်ပဲဖြစ်ပါတယ်။ CSS variables တွေကိုအရင်က custom Properties လို့သတ်မှတ်ကြပါတယ်။ဘာ့ကြောင့်လဲဆိုတော့အဲ့တာတွေက regular CSS properties တွေနဲ့ပုံစံတူပြီးတော့ custom property notation လို့ခေါ်တဲ့ special syntex ကိုအသုံးပြုပါတယ်။ property name ရဲ့ရှေ့မှာ dashes နှစ်ခုရှိတာက custom property ဆိုတာကို browser ကိုပြောပြပါတယ်။ အဲ့တာကြောင့်ကျန်တော်တို့က dashed နှစ်ခုနဲ့စရေးပြီး property တွေကိုနာမ်မည်ပေးနိုင်ပါတယ်။အဲ့တာကို technically အရတော့ custom properties လို့ခေါ်ပြီးတော့ developer တွေအများစုကတော့ CSS Variables လို့ခေါ်ပါတယ်။CSS variables တွေကိုအသုံးပြုခြင်းက ကျွန်တော်တို့ရဲ့ stylesheet ထဲမှာရှိတဲ့ variables တေါထပ်ခါထပ်ဖြစ်တာတွေကိုလျော့ချပေးပါတယ်။(--property-name: value;) ဆိုပြီးရေးနိုင်ပြီးတော့ ကျွန်တော်တို့ရဲ့code တွေထဲကနေရည်ညွန်းချင်တယ်ဆိုရင်တော့ var(--property-name) ဆိုတာကိုအသုံးပြုနိုင်ပါတယ်။ပြီးတော့ css variables တွေက game changer ဖြစ်ပြီးတော့ preprocessor variable တွေထက်ပိုပြီးပြုလုပ်နိုင်ပါတယ်။

  • Decalring & Using CSS Variables

    CSS variables တွေကိုအသုံးပြုတဲ့အခါမှာကျွန်တော်တို့ရဲ့ stylesheet ထဲမှာ process အဆင့်နှစ်ခုရှိပါတယ်။ပထမတစ်ခုကတော့ custom property notation တွေကိုအသုံးပြုပြီးတော့ selector ထဲမှာ variable တွေကိုကြေငြာရမှာပဲဖြစ်ပါတယ်။variable သို့ property name တွေကိုရည်ညွှန်းတဲ့အခါမှာတော့ var() ဆိုတဲ့ function ကိုအသုံးပြုပါတယ်။ အောက်ပါဥပမာထဲမှာ CSS variables တွေကိုကြေငြာနည်းနဲ့ပြန်ခေါ်ပြီးအသုံးပြုပုံတွေကိုထပ်ပြီးလေ့လာနိုင်ပါတယ်။ CSS variables တွေက preprocessor variables တွေနဲ့တူညီအောင်မတည်ဆောက်နိုင်ပါဘူး။ဒါပေမယ့်တူညီတဲ့ functions တွေတော့တွေ့ရပါတယ်။CSS variables တွေက things အများကြီးကိုပြုလုပ်နိုင်ပြီးတော့ preprocessor variables တွေကတော့မလုပ်နိုင်ပါဘူး။preprocessors တွေအလုပ်လုပ်တာကိုကျွန်တော်တို့ browser ထဲမှာမြင်ရမှာမဟုတ်ပါဘူး။CSS တွေကို compile လုပ်တဲ့အချိန်မှာ preprocessors တွေက variables တွေကို static values တွေနဲ့အစားထိုးတာပဲဖြစ်ပါတယ်။သူတို့က DOM structure တွေကိုသတိမထားပါဘူး။အခြားတစ်ဖက်မှာတော့ css variables တွေက flexibles ပိုဖြစ်ပါတယ်။ဘာ့ကြောင့်လဲဆိုတော့အဲ့တာတွေကို browser က runtime ပြုလုပ်တဲ့အချိန်မှာ update လုပ်ပေးလို့ပဲဖြစ်ပါတယ်။css variables တွေမှာ dynamic capabilities တွေရှိပါတယ်။preprocessor variables တွေကိုကျွန်တော်တို့ inline stylesheet တွေကနေလည်းခေါ်ပြီးအသုံးပြုနိုင်ပါတယ်။ 

    
/* VARIABLES ----------------------------- */

:root {
    --color-primary: #278da4;
    --color-secondary: #b13c69;
    --color-bg: #3acec2;
    --color-bg-light: #009fe1;
    --gradient: var(--color-bg-light), var(--color);
    --font-stack-primary: 'Raleway', sans-serif;
    --font-stack-secondary: 'Bree Serif', serif;
}


/* BASE STYLES --------------------------- */

body {
    font-family: var(--font-stack-primary);
}

h2 {
    font-family: var(--font-stack-secondary);
}

a,
h2 {
    color: var(--color-primary);
}

    var functions တွေကနောက်ထပ် arguments တွေကိုလက်ခံနိုင်ပြီး fall back value တွေကဲ့သိ့ုပြုမူလုပ်ဆောင်ပါတယ်။second argument က variable အတွက်အစားထိုးပြီးအသုံးပြုချင်းဖြစ်ပါတယ်။fall back values တွေကိုအသုံးပြုပြီး ကျွန်တော်တို့ element အတွက် defensive styles တွေကိုရေးနိုင်ပါတယ်။နောက်ထပ်ကျွန်တော်တို့ရဲ့ stylesheet ကိုလည်းပဲ sharable နဲ့ extensible တွေပြုလုပ်နိုင်ပါတယ်။CSS variables တွေအသုံးပြုပြီးတော့လည်းကျွန်တော်တို့ကရိုးရှင်းတဲ့တွက်ချက်မှုတွေပြုလုပ်နိုင်ပါတယ်။အဲ့လိုတွက်ချက်ဖို့အတွက်ကတော့ကျွန်တော်တို့က CSS calc function တွေအသုံးပြုနိုင်ပါတယ်။ 

    
                                    
header {
    background: linear-gradient(var(--gradient)),url('..img/bg.jpg') no-repeat;
    background-blend-mode: multiply;
    background-size: cover;
    margin-bottom: calc(var(--spacing)* 1em); /* 2em */
}

/* COMPONENT STYLES -------------------------*/

.main-nav li {
    margin: 0 calc(var(--spacing)* 10px);
}

  • SVG Images width Variables

    element ရဲ့ style attributes အတွင်းမှာ update နဲ့ altered လုပ်နိုင်တာတွေကိုကျွန်တော်တို့လေ့လာခဲ့ပြီးဖြစ်ပါတယ်။အဲ့လိုပါပဲကျွန်တော်တို့က style တွေကိုပြုပြင်ဖို့ရန် နဲ့ svg တွေထဲမှာ presentation attributes တွေကိုပြုပြင်ဖို့ရန်အတွက် CSS variables တွေကိုအသုံးပြုနိုင်ပါတယ်။ဒီတစ်ခာမှာတော့ကျွန်တော်တို့ SVG icons တွေကို CSS variables တွေအသုံးပြုပြီးလွယ်ကူစွာတည်ဆောက်နိုင်တာကိုလေ့လာသွားမှာပဲဖြစ်ပါတယ်။အောက်ပါ ဥပမာထဲမှာတော့ svg image တွေကိုကျွန်တော်တို့ fill ဆိုတာကိုကြေငြာပြီးအသုံးပြုပြထားတာဖြစ်ပါတယ်။နောက်ထပ်ကျွန်တော်တို့ var ကို values အဖြစ်အသုံးပြုနိုင်ပြီး အခြား svg attributes ဖြစ်တဲ့ stroke လိုမျိုး attribute ပဲဖြစ်ပါတယ်။

  • CSS Variables with JavaScript

    CSS variables တွေက cascade နဲ့ inheritance တွေကိုသုံးပြီးတော့ update မလုပ်ပါဘူး။ကျွန်တော်တို့တွေက javascript တွေကနေပြီးတော့ CSS variables တွေကို read , write (သို့) get , set တွေပြုလုပ်နိုင်ပါတယ်။get PropertyValue ကတော့ custom property ရဲ့ value တွေကိုရယူဖို့အတွက်အသုံးပြုပြီးတော့ setProperty method ကိုတော့ value တွေကိုထားဖို့အတွက်အသုံးပြုတာပဲဖြစ်ပါတယ်။ထပ်ပြီးလေ့လာချင်တယ်ဆိုရင်တော့ ကျွန်တော်တို့ CSS Custom Properties (CSS Variables) Sample မှာသွားပြီးလေ့လာနိုင်ပါတယ်။အဲ့ဒီထဲမှာ CSS code snipped တွေနဲ့ javaScript snipped တွေကိုပါဖော်ပြပေးထားပါတယ်။အောက်ပါ ဥပမာထဲမှာတော့ CSS variables တွေကို JavaScript တွေနဲ့တွဲပြီးအသုံးပြုထားတဲ့ page ကိုဖော်ပြထားတာပဲဖြစ်ပါတယ်။ var function အလုပ်လုပ်တဲ့အခါမှာ custom properties တွေကဘယ်နေရာမှာအလုပ်လုပ်တယ် ဘယ်နေရာမှာအလုပ်မလုပ်ဘူးဆိုတာကိုသိသင့်နေပြီဖြစ်ပါတယ်။Selector ထဲမှာအသုံးပြုတဲ့အခါမှာတော့ var function တွေက invalid ဖြစ်ပါတယ်။var တွေကိုကျွန်တော်တို့ CSS values တွေထဲမှာပဲအသုံးပြုနိုင်မှာဖြစ်ပါတယ်။နောက်ထပ်ကျွန်တော်တို့ Media query expression ထဲမှာလည်း var တွေကိုအသုံးပြုနိုင်မှာမဟုတ်ပါဘူး။နောက်ဆုံးကျွန်တော်တို့ length value တွေကိုတည်ဆောက်ဖို့အတွက်တောင်မှကျွန်တော်တို့ var function တွေကိုအသုံးမပြုနိုင်ပါဘူး။ Variable names တွေက case sensitive ဖြစ်နိုင်ပါတယ်။CSS Variables တွေကအသုံးဝင်ပြီးတော့ ကောင်းမွန်တဲ့ feature တစ်ခုပဲဖြစ်ပါတယ်။
