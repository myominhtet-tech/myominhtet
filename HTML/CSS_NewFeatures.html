  • Filters

    ဒီတစ်ခါမှာတော့ကျွန်တော်တို့ CSS တွေရဲ့နောက်ဆုံးထွက်ထားတဲ့ features တွေကိုလေ့လာသွားမှာဖြစ်ပါတယ်။အဲ့တာတွေကတော့ filter effects ,Calc function, flex block နဲ့ multi-column layout တွေပဲဖြစ်ပါတယ်။ပထမဆုံးကျွန်တော်တို့ filter effects တွေကိုစပြီးလေ့လာသွားမှာဖြစ်ပါတယ်။အဲ့တာက photoshop တွေလိုပဲပြုလုပ်နိုင်တာတွေပါဝင်ပါတယ်။HTML content တွေကိုရိုးရှင်းတဲ့ CSS syntax တွေနဲ့သတ်မှတ်တာဖြစ်ပြီးတော့ blurs နဲ့ drop shadows တွေလိုမျိုးဖြစ်ပါတယ်။အခြား effects တွေကတော့ gray scale နဲ့ contrast adjustments တွေပဲဖြစ်ပါတယ်။filter တွေက DOM elements တွေတိုင်းကိုသတ်မှတ်နိုင်ပါတယ်။images,texts တွေနဲ့ video နဲ့ canvas elements တွေကိုတောင်မှာ apply ပြုလုပ်နိုင်ပါတယ်။filter တွေက parameter တွေပေါ်မှာအခြေခံပြီးတော့browser ရဲ့မြင်ကွင်းပေါ်မှာအခြေခံပြီးတော့pixel လေးတွေပေါ်မှာပြောင်းလဲပေးတာဖြစ်ပါတယ်။အဲ့ဒီ filter တွေက browser ပေါ်မှာအခြေခံတဲ့ webkit တွေပေါ်မှာအလုပ်လုပ်တာပဲဖြစ်ပါတယ်။Chrome နဲ့ Safari တွေလိုမျိုးပဲဖြစ်ပါတယ်။အဲ့တာကြောင့်ကျွန်တော်တို့က webkit prefix ကိုအသုံးပြုပေးရမှာပဲဖြစ်ပါတယ်။ပထမအနေနဲ့ကျွန်တော်တို့ sepia filter ဆိုတာကိုလေ့လာကြည့်ပါမယ်။အဲ့တာကတော့ image တွေကို sepia tone တွေနဲ့ပြုလုပ်ပါတယ်။အရင်ခေတ်တွေကold western videoတွေလိုမျိုးသို့မဟုတ် antique photograph တွေလိုမျိုးကျွန်တော်တို့မြင်ရမှာပဲဖြစ်ပါတယ်။ဥပမာထဲမှာကျွန်တော်တို့ sepia ရေးနည်းကိုဖော်ပြထားပါတယ်။sepia ကpercentage တွေကိုပဲလက်ခံနိုင်တာပဲဖြစ်ပါတယ်။ နောက်ထပ် filter တွေဖြစ်တဲ့ grapscale,blur,opacity,huge-rotate,drop-shadow ဆိုတာတွေကိုကျွန်တော်တို့ ဥပမာထဲမှာတစ်ခုချင်းစီကိုသေချာရေးပြထားတာပဲဖြစ်ပါတယ်။တစ်ချို့ browser တွေမှာတော့အဲ့တာတွေကအလုပ်မလုပ်နိုင်ပဲရှိပါလိမ့်မယ်အဲ့တာကြောင့်ကျွန်တော်တို့ကဘယ် browser တွေ version ဘယ်လောက်တွေမှာပဲထောက်ပံ့ပေးတယ်ဆိုတာကို---ဒီမှာ---သွားကြည့်နိုင်ပါတယ်။

  • Calc Function

    နောက်ထပ်ကျွန်တော်တို့ Calc function ကိုလေ့လာသွားမှာပဲဖြစ်ပါတယ်။Calc CSS method ကတော့ length တွေအတွက် value တွေကိုတွက်နိုင်ပြီး mathematical expressions တွေကိုအသုံးပြုတာပဲဖြစ်ပါတယ်။ကျွန်တော်တို့က html elements တွေရဲ့ size တွေကို size unit တွေရောပြီးအသုံးပြုနိုင်ပါတယ်။အဲ့တာကိုကျွန်တော်တို့ အသုံးပြုတဲ့အခါမှာ browser ကကျွန်တော်တို့ကိုမှန်ကန်တဲ့ sizes တွေထွက်လာအောင်အလုပ်လုပ်ပေးပါလိမ့်မယ်။ဥပမာထဲမှာ calc css အသုံးပြုပုံကိုကျွန်တော်တို့ဖော်ပြထားတာပဲဖြစ်ပါတယ်။ 

    
.wrapper {
    background: #1f90ff;
    height: -webkit-calc(80% - 100px);    /*This calculate value of height.*/
    width: -webkit-calc(100%/3 - 2em);
}

  • New Flexbox Features

    Flex Box feature သို့မဟုတ် Flexible Layout တွေကိုထပ်ပြီးလေ့လာသွားမှာဖြစ်ပါတယ်။CSS ကlayout တွေကိုထိန်းချုပ်နိုင်တဲ့ တိကျတဲ့language တစ်ခုတော့မဟုတ်ပါဘူး။အဲ့တာကြောင့်ကျွန်တော်တို့က feature အသစ်နဲ့ flex-box ကိုအသုံးပြုမှာပဲဖြစ်ပါတယ်။ကျွန်တော်တို့ layout တွေအတွက်ပိုပြီးလုပ်လောက်ဖို့အတွက်နှင့် powerful primitive ဖြစ်တဲ့ flex box features တွေကိုအသုံးပြုသွားမှာပဲဖြစ်ပါတယ်။Flex Box တွေက 1-dimensional CSS layout တွေကိုထိန်းပေးတဲ့နေရာမှာကောင်းမွန်ပါတယ်။ဘာ့ကြောင့်လဲဆိုတော့ဘယ်လို direction မျိုးမှာမဆိုကျွန်တော်တို့က elements တွေရဲ့ Layout တွေကိုကောင်းမွန်လှပအောင်ပြုပြင်ပေးနိုင်လို့ပဲဖြစ်ပါတယ်။ဥပမာထဲမှာ flex box features တွေဘယ်လိုအလုပ်လုပ်သလဲဆိုတာကိုလေ့လာနိုင်ပါတယ်။ 

    
ul {
    display: -webkit-flex;
    -webkit-flex-flow: row;
    -webkit-flex-wrap: wrap;
}

li {
    -webkit-flex-grow: 1;
    height: 300px;
}

li:nth-child(3) {
    -webkit-flex-grow: 5;
    -webkit-order: -1;
    height: 75px;
    -webkit-align-self: center;
}

  • Blend-Mode

    ဒီတစ်ခါမှာတော့ကျွန်တော်တို့တွေ graphic editor, တွေထဲမှာအများဆုံးအသုံးပြုတဲ့ tools နဲ့တူညီတဲ့ blend modes ကိုလေ့လာမှာပဲဖြစ်ပါတယ်။CSS blend mode တွေက တစ်ခုသို့မဟုတ်တစ်ခုထက်ပိုတဲ့ layer တွေကိုအတူတကွရောနှောပြီးတော့သတ်မှတ်နိုင်တာဖြစ်ပြီးအမှန်တကယ်တည်ရှိတဲ့ visual effects တွေကိုလည်းဖန်တီးနိုင်ပါတယ်။အဲ့ဒီ blending effects တွေကိုမကြာခနဆိုသလို graphic design သို့မဟုတ် digital image editing တွေမှာအသုံးပြုပါတယ်။ဒါပေမယ့် CSS blend modes တွေကိုအသုံးပြုပြီးတော့ကျွန်တော်တို့က browser ထဲမှာတူညီတဲ့ bleding effects တွေကိုပြန်ပြီးတည်ဆောက်နိုင်ပါတယ်။blend mode တွေဟာပေါင်းစပ်ခြင်းတွေ တူညီခြင်းတွေနဲ့ CSS Specification တွေရဲ့အစိတ်အပိုင်းတွေပဲဖြစ်ပါတယ်။composing တွေ blemding တွေကဘယ်လိုအလုပ်လုပ်သလဲ။ကောင်းပြီ! compositing ဆိုတာကတော့ elements တစ်ခုက၎င်းရဲ့ backdrop နဲ့ပေါင်းစပ်ပြီးအလုပ်လုပ်တာပဲဖြစ်ပါတယ်။ပြီးတော့ backdrop က element ရဲ့နောက်မှာရှိတဲ့ content ပဲဖြစ်ပါတယ်။blending ဆိုတာကတော့ color တွေရောတဲ့အခါမှာတွက်ချက်ပြီးတော့ form တွေကို compositing တွေပဲဖြစ်ပါတယ်။ကျွန်တော်တို့တွေက CSS တွေနဲ့ html element တွေရဲ့ background layer ကိုအခြား layer နဲ့ blend လုပ်နိုင်ပါတယ်။ background layer က image တွေ gradient တွေသို့မဟုတ် background color တွေဖြစ်နိုင်ပါတယ်။CSS မှာ blend elements တွေကို blend modes တွေနဲ့အသုံးပြုဖို့အတွက် properties နှစ်မျိုးရှိပါတယ်။အဲ့တာတွေကတော့ background-blend-mode နဲ့ mix-blend-mode ပဲဖြစ်ပါတယ်။ background-blend-mode နဲ့ mix-blend-mode မှာတူညီတဲ့ value တွေ ၁၆ မျိုးဆီရှိပါတယ်။အဲ့တာတွေကို background-blend-modeမှာသွားပြီးလေ့လာနိုင်ပါတယ်။ 

    
                                

/* BLEND MODE */

body {
    font: 1em/1.4 'Helvetica Neue', sans-serif;
    color: #eee;
    background: url('falls.jpg') no-repeat, linear-gradient(#6de590, #e67478);
    background-blend-mode: multiply;
    background-size: cover;
}

h1 {
    font-size: 12em;
    margin: 0;
    line-height: 1;
    letter-spacing: -2rem;
    color: #6b845d;
}

span:nth-child(2) {
    color: #801b3a;
}

span:nth-child(3) {
    color: #164d67;
}

h1 span {
    mix-blend-mode: screen;
}

  • Tooltips width Pure CSS

    အခုတစ်ခါမှာတော့ကျွန်တော်တို့ tooltips တွေအကြောင်းကိုပြောပြသွားမှာဖြစ်ပါတယ်။ HTML နဲ့ CSS တွေကိုပဲအသုံးပြုသွားမှာဖြစ်ပြီး tooltips တွေကအသုံးပြုသူတွေကို informations အပိုတွေထပ်ပေးဖို့အတွက်အကောင်းဆုံးနည်းလမ်းတွေပဲဖြစ်ပါတယ်။အဲ့တာတွေကအသုံးပြုတာအများဆုံး design pattern တွေဖြစ်ပြီးများစွာသော popular front end frameworks တွေမှာတွေ့နိုင်ပါတယ်။ဥပမာ Bootstrap , Foundation တွေပဲဖြစ်ပါတယ်။အဲ့ဒီ frameworks တွေနှစ်ခုစလုံးမှာ javaScript ကိုအသုံးပြုပြီးတော့ frameworks တွေကိုကိုင်တွယ်ပါတယ်။ ဒါပေမယ့်အဲ့တာတွေက css တွေကိုအသုံးပြုပြီးလွယ်ကူစွာတည်ဆောက်နိုင်ပါတယ်။အသုံးပြုတဲ့နည်းလမ်းတွေကတော့ pseudo-elements တွေနဲ့ ကျွန်တော်တို့အသုံးပြုချင်တဲ့ data attributes တွေကိုအသုံးပြုတာပဲဖြစ်ပါတယ်။ပထမ data attribute ကကျွန်တော်တို့ရဲ့ tooltip message ကို html ထဲမှာသိမ်းထားပေးပါလိမ့်မယ်။tooltip အလုပ်လုပ်တဲ့အချိန်မှအဲ့တာကဖော်ပြပေးပါတယ်။tooltips တွေကိုထည့်ဖို့အတွက်ကျွန်တော်တို့က paragraph ထဲမှာ span element တွေကိုအသုံးပြုရပါတယ်။နောက်ထပ် span tag ထဲမှာတော့ custom data attributes လို့ခေါ်တဲ့ data-tip တွေကိုအသုံးပြုပါတယ်။အဲ့တာကတော့အသုံးပြုတဲ့သူတွေကိုဖော်ပြတဲ့စာသားတွေကိုထည့်ရေးပေးရတာဖြစ်ပါတယ်။tooltips တွေကကျွန်တော်တို့ mouse hover ပြုလုပ်တဲ့အခါမှာပေါ်လာမှာဖြစ်ပါတယ်။အောက်ပါဥပမာထဲမှာကျွန်တော်တို့ tooltips နှစ်ခုကိုဖော်ပြထားတာပဲဖြစ်ပါတယ်။
    Tooltip.css 

    
    <!DOCTYPE html>
    <html>
        <head>

            <meta name="viewport" content="width=device-width, initial-scale=1">

            </div>
        </body>
    </html>
