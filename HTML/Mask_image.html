  • CSS Masks

    ဒီတစ်ခါမှာတော့ကျွန်တော်တို့ CSS design effect တွေကို cover ပြုလုပ်ဖို့အတွက် CSS mask တွေကိုလေ့လာသွားမှာပဲဖြစ်ပါတယ်။အဲ့တာကြောင့်ကျွန်တော်တို့က masking နဲ့ clipping တွေကိုအများဆုံးအသုံးပြုတဲ့ features တွေဖြစ်ပြီး photoshop တွေ Sketch တွေနဲ့ပုံစံတူတဲ့ graphic editing tools တွေပဲဖြစ်ပါတယ်။ကျွန်တော်တို့က အဲတာတွေကို elements တွေရဲ့မြင်နိုင်တဲ့အပိုင်းတွေကို hide ထားဖို့အတွက်အသုံပြုပါတယ်။ဒါပေမယ့်ကျွန်တော်တို့က mask တွေကို browser ပေါ်မှာCSS တွေနဲ့တိုက်ရိုက်တန်းပြီးအသုံးမပြုနိင်ပါဘူး။design effects အချို့နဲ့အဲ့တာကို create ပြုလုပ်နိုင်ပါတယ်။Masking က CSS masking specification ရဲ့အပိုင်းတစ်ခုပဲဖြစ်ပါတယ်။method နှစ်ခုရှိပြီးတော့ အစိတ်အပိုင်းလေးတွေအတွက်သို့မဟုတ် html elements တစ်ခုလုံးကိုဖျောက်ထားနိုင်ပါတယ်။အဲ့ဒီ method တွေကတော့ masking နဲ့ clipping တွေပဲဖြစ်ပါတယ်။ masking feature တွေကသေချာတဲ့ areas တွေကို masking out ပြုလုပ်ခြင်းဖြင့် elements တွေရဲ့အပိုင်းတွေကိုမှေးမှိန်သွားအောင်ပြုလုပ်နိုင်ပါတယ်။ အဲ့တာကြောင့်ကျွန်တော်တို့က elements တွေကို masking ပြုလုပ်ချင်တယ်ဆိုရင်တော့ mask image တွေကိုအသုံးပြုပြီး elements တွေရဲ့မြင်နိုင်တဲ့အပိုင်းတွေကို filter ပြလုပ်နိုင်ဖို့ပဲဖြစ်ပါတယ်။mask image တွေမှာ တစ်ပိုင်းကအနက်ရောင်လုံးလုံးဖြစ်နေပြီးတစ်ပိုင်းကcompletely transparent ဖြစ်နေတယ်ဆိုရင်အဲ့တာမျိုးကို Alpha Mask လို့ခေါ်နိုင်ပါတယ်။နောက်ထပ် mask image တွေအတွက်ကျွန်တော်တို့ CSS ထဲမှာwebkit vendor prefix ကိုအသုံးပြုဖို့လိုအပ်ပါတယ်။CSS masks တွေကအဆင်သင့်ပြီး flexible ဖြစ်တဲ့ design features တွေပဲဖြစ်ပါတယ်။ကျွန်တော်တို့ website တွေမှာရှိတဲ့ headlines , buttons , icons သို့မဟုတ်အခြားအစိတ်အပိုင်းတွေမှာကျွန်တော်တို့အသုံးပြုနိုင်ပါတယ်။Internet Explorer မှာတော့ mask image property ကိုထောက်ပံ့ပေးမှာမဟုတ်ပါဘူး။
