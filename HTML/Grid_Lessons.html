ပေါ်မှာ speakers list တွေကိုတည်ဆောက်တော့မယ်ဆိုရင် content တွေမှာ CSS grid တွေနဲ့ Layout တွေရဲ့အရေးကြီးတဲ့အပိုင်းတွေကိုပြုလုပ်ပါတယ်။ဒါမှမဟုတ် သေးငယ်တဲ့ responsive forms သို့ မဟုတ် media objects   • Grid Concept

    CSS grid တွေနဲ့ layout တွေကိုမတည်ဆောက်ခင်မှာကျွန်တော်တို့ဟာ အဲ့တာတွေရဲ့ concept နဲ့ terminology အချိုကိုနားလည်ထားဖို့လိုအပ်ပါတယ်။CSS Grid layout တွေမှရှိတဲ့အရေးကြီးတဲ့ elements နှစ်ခုကတော့ grid containers နဲ့ grid itemsတွေပဲဖြစ်ပါတယ်။ child elements တွေဟာ space တွေ sized တွေ aligned တွေဟာဘယ်လိုတိုက်ရိုက်သတ်ရောက်သလဲဆိုတာmanage ပေးပါတယ်။ပြီးတော့ အဲ့ဒီ grad container တွေက grid formatting context တွေကိုတည်ဆောက်ပေးပါတယ်။Grid container တွေကိုတည်ဆောက်တဲ့အခါ element ရဲ့ direct child တွေက grid items တွေဖြစ်လာပါတယ်။ဥပမာကျွန်တော်တို့အောက်ပါ Example ထဲမှာရှိတဲ့ main element တွေဟာ grid container တွေဖြစ်ပြီးတော့ article တွေ heading တွေ paragraph တွေကတော့ grid item တွေပဲဖြစ်ပါတယ်။grid formatting context တည်ဆောက်တဲ့အခါမှာကျွန်တော်တို့ ကျွန်တော်တို့ုက grid ထဲမှာ content တွေကို positing , aligning တွေစပြီးပြုလုပ်နိုင်ပါတယ်။အဲ့ဒီနောက် grid ကဒေါင်လိုက် အလျှားလိုက်လိုင်းတွေထည့်ပေးမှာဖြစ်ပြီး အဲ့တာတွေကို grid lines လို့ခေါ်ပါတယ်။grid lines တွေဟာ grid container ထဲမှာရှိတဲ့ space တွေကို column တွေ rows တွေအဖြစ်သတ်မှတ်ထားပါတယ်။အလျှားလိုက် grid lines တွေကတော့ rows တွေကိုဖော်ပြပြီးတော့ ဒေါင်လိုက် grid lines တွေကတော့ columns တွေကိုကိုယ်စားပြုပါတယ်။grid line တစ်ခုက column သို့ row တစ်ခုရဲ့ အခြား side ပေါ်မှာတည်ရှိနေပါတယ်။

  • Grid Container

    ဒီတစ်ခါမှာတော့ကျွန်တော်တို့ ပထမအဆင့်အနေနဲ့ grid ကိုတည်ဆောက်ရမှာပဲဖြစ်ပါတယ်။grid containerကိုသတ်မှတ်ပြီးတော့ column တွေ row တွေကိုလည်းပြုလုပ်ပေးရမှာဖြစ်ပါတယ်။ဒီတစ်ခါမှာတော့ ကျွန်တော်တို့ grid တွေနဲ့ချိတ်ဆက်နေတဲ့ properties တွေကိုဖော်ပြမှာဖြစ်ပါတယ်။column တွေကိုသတ်မှတ်ဖို့အတွက် grid-template-column ကိုအသုံးပြုပြီးတော့ row တွေသတ်မှတ်ဖို့အတွက် grid-template-row ကိုအသုံးပြုပါတယ်။grid တွေကိုရေးဖို့အတွက် CSS file အသစ်ကိုဆောက်ပြီးရေးပေးရမှာပဲဖြစ်ပါတယ်။CSS grid layout ကို grid container တွေနဲ့စပြီးတည်ဆောက်ရပါတယ်။ဥပမာထဲမှာပြထားသလိုပါပဲ browser မှာ grid ကိုဖော်ပြဖို့အတွက် display ဆိုတဲ့ properties ကိုသုံးပြီးတော့ grid ဆိုတဲ့ value ကိုထည့်ပေးရမှာဖြစ်ပါတယ်။အဲ့ဒီအခါမှာမှ အတွင်းထဲမှာရှိတဲ့ item တွေက grid items တွေဖြစ်လာမှာပဲဖြစ်ပါတယ်။grid container တွေက grid item တွေကိုcoiumn တစ်ခုထဲမှာ ထည့်ပေးထားမှာဖြစ်ပြီးအဲ့တာဟာ default style ပဲဖြစ်ပါတယ်။CSS grid layout ထဲမှာ column တွေ row တွေ(သို့) grid lines နှစ်ခုကြားမှာရှိတဲ့ space တွေဟာ grid tracks တွေကိုရည်ညွှန်းပါတယ်။column tracks တွေကိုထားဖို့အတွက် grid-template-columns ကိုသုံးပြီး row tracks တွေကိုထားဖို့အတွက်တော့ grid-template-rows တွေကိုသုံးပါတယ်။အဲ့ဒီ properties တွေအားလုံးကို grid container ထဲမှာရေးနိုင်ပါတယ်။ grid container CSS ruke တွေက grid layout configuration rule တွေနဲ့တူပါတယ်ဘာ့ကြောင့်လဲဆိုတော့ grid ထဲမှာဖြစ်နေတာတွေအားလုံးကို rule တစ်ခုတည်းကနေပြုပြင်နိုင်ပါတယ်။ကျွန်တော်တို့အောက်ပါဥပမာထဲမှာ grid-container တွေ grid-template-column တွေကိုကောအသုံးပြုပြထားပါတယ်။ The following code is two css files.

    Grid မှာ default layout က mobile devices (သို့) tablet တွေမှာကြည့်ဖို့အတွက်အဆင်ပြေနိုင်ပါတယ်။အဲ့တာကြောင့် defaultပုံစံ layout ကိုပြန်ပြောင်းခဲ့ရပါမယ်။နောက်ထက်ကျွန်တော်တို့ဟာ media query တွေကိုအသုံးပြုပြီးတော့ grid တွေကိုချိန်ညှိရပါမယ်။အဲ့တာကြောင့် media query တစ်ခုကိုတည်ဆောက်ပြီး viewport ကို 768 pixels (or) wider ဆိုပြီးဥပမာထဲမှာလိုထားခဲ့ပေးပါမယ်။အဲ့ဒီ media query ထဲမှာ container ကိုပြန်သုံးပြီးကျွန်တော်တို့ပေါ်ချင်တဲ့ layout ပုံစံကိုတည်ဆောက်ခဲ့ရပါမယ်။နောက်ထပ် media query တစ်ခုကိုတည်ဆောက်ပြီး width 992px (or) wider လို့ပေးခဲ့ပြီးကျွန်တော်တို့ပြောင်းချင်တဲ့ layout ပုံစံကိုပြောင်းပေးနိုင်ပါတယ်။
