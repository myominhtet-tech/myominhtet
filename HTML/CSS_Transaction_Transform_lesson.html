    LEARNING TRANSACTIN & TRANSFORM

  • Introduction

    အခုတစ်ခါကျွန်တော်တို့ထပ်ပြီးစိတ်လှုပ်ရှားစရာကောင်းတဲ့ feaures နှစ်ခုကိုထပ်ပြီးလေ့လာသွားမှာဖြစ်ပါတယ်။Developers တွေက websites နဲ့ apps တွေရဲ့ပြုမူလုပ်ဆောင်မှုကိုတိုးမြင်လုပ်ဆောင်တဲ့နေရာတွေနဲ့ ရိုးရှင်းတဲ့ animationsတွေတည်ဆောက်တဲ့အခါမှာအများဆုံးအသုံးပြုကြပါတယ်။အဲ့ဒီတော့ကျွန်တော်တို့ browser ထဲမှာ webpages တွေကို CSS တွေနဲ့ဘယ်လိုထိန်းချုပ်လဲဆိုတာကိုကျွန်တော်တို့သိရှိပြီးဖြစ်ပါတယ်။CSS Transitions နဲ့ transforms တွေနဲ့ဆိုရင်ကျွန်တော်တို့content တွေကို styles တွေတစ်ခုနဲ့တစ်ခုကိုပြောင်းတဲ့ animations ကိုထည့်တဲ့နေရာတွေမှာအသုံးပြုနိုင်ပါတယ်။CSS transitions features တွေကိုအသုံးပြုပြီးပေးထားတဲ့အချိန်အတွင်းမှာ CSS properties တွေကို value တစ်ခုမှ အခြားတစ်ခုကိုပြောင်းလဲနိုင်ပါတယ်။ကျွန်တော်တို့ရဲ့ webpage ပေါ်မှာ mouse တင်တဲ့အခါမှာပြောင်းလဲသွားတာကိုပြုလုပ်ပေးတာပဲဖြစ်ပါတယ်။CSS transforms တွေကိုအသုံးပြုတဲ့အခါမှာတော့ကျွန်တော်တို့က elements တွေရဲ့တည်နေရာတွေကိုပြောင်းလဲနိုင်ပြီး ပုံမှန် document flow ကိုမထိခိုက်စေပဲနဲ့သူ့ရဲ့shape ကို alert ပြုလုပ်နိုင်ပါတယ်။နောက်ထပ်ကျွန်တော်တို့ကတစ်ခုနဲ့တစ်ခုထက်ပိုတဲ့ transform functions တွေကိုအသုံးပြုပြီးတော့ content တွေကို move, rotate, scale and skew ပြုလုပ်နိုင်ပါတယ်။ Transitions တွေ transforms တွေကအမှန်တကယ်တော့ချိတ်ဆက်နေတဲ့ CSS features တွေမဟုတ်ပါဘူး။ဆိုလိုတာကတော့သူတို့တွေက stand alone တည်ရှိနိုင်တာတွေပဲဖြစ်ပါတယ်။ကျွန်တော်တို့တွေက transform တွေမပါပဲနဲ့ transitionတွေကိုတည်ဆောက်နိုင်သလို transition တွေမပါပဲနဲ့လည်း transformတွေတည်ဆောက်နိုင်ပါတယ်။ဒါပေမယ့်အကောင်းဆုံးအပိုင်းကတော့ CSS transforms တွေကို CSS transitions တွေကိုပေါင်းပြီးတည်ဆောက်ခြင်းပဲဖြစ်ပါတယ်။

  • Transition Property

    CSS transitions တွေကကျွန်တော်တို့ရဲ့ Interface ကို animate လုပ်တဲ့အပိုင်းတွေမှာလွယ်ကူအောင်ပြုလုပ်ပေးပါတယ်။သတ်မှတ်လိုက်တဲ့အချိန်အတွင်းမှာvalue တစ်ခုကနေတစ်ခုကို CSS properties တွေအစုလိုက်ပြောင်းလဲဖို့လုပ်ဆောင်ပေးပါတယ်။အဲ့လိုလုပ်တဲ့အခါမှာကျွန်တော်တို့တွေက browser တွေကိုအသိပေးဖို့လိုအပ်ပါတယ်။ဘယ် properties က transition လုပ်ဖို့အတွက်ဆိုတာနဲ့ transition လုပ်ဖို့အချိန်ဘယ်လောက်ယူမယ်ဆိုတာကိုတွေကိုအသိပေးဖို့ပဲဖြစ်ပါတယ်။Transition တွေနှောက်နေးဖို့ရန်ကြာဖို့ရန်ကျွန်တော်တို့က browser ကိုပြသလုပ်ဆောက်နိုင်ပါတယ်။ပြီးတော့အဲ့ဒီအချိန်အတိုင်းအတာအတွင်းမှာ CSS lines အချို့ကိုအသုံးပြုပြီးတော့transition တွေဘယ်လိုပြောင်းလဲတယ်ဆိုတယ် speed ကိုလည်းထိန်းချုပ်နိုင်ပါတယ်။user's action တွေသို့မဟုတ်ပြောင်းလဲနေတဲ့ state တွေဖြင့် transitions တွေကအမြဲတမ်းဖြစ်ပေါ်နေပါတယ်။transitions တွေဖြစ်ပေါ်ဖို့်အတွက်နည်းလမ်းအချို့ရှိပြီးတော့ javascript ကိုသုံးပြီးလည်းကျွန်တော်တို့ transitionလုပ်နိုင်ပါတယ်။transition property ကိုအသုံးပြုတဲ့အခါမှာ start value နဲ့ end value ဆိုပြီးကျွန်တော်တို့လိုအပ်ပါတယ်။CSS transitions တွေနဲ့ချိတ်ဆက်နေတဲ့မတူညီတဲ့ properties ၄မျိုးရှိပါတယ်။တစ်ခုကတော့ property တွေကိုသတ်မှတ်တယ်သို့မဟုတ် transition လုပ်ဖို့ရန် properties အစုတွေကိုသတ်မှတ်တာပဲဖြစ်ပါတယ်။နောက်တစ်ခုကတော့ transition duration ဖြစ်ပြီးတော့ timing function ကိုသတ်မှတ်ဖို့အတွက် properties နှစ်ခုလိုအပ်ပါတယ်။အဲ့တာကတော့ transition အတွက်လိုအပ်တဲ့ property ပဲဖြစ်ပါတယ်။transition duration က numeric value ကိုလက်ခံပြီးတော့ CSS time unit တွေလိုက်တာပဲဖြစ်ပါတယ်။time unit က second သို့မဟုတ် milliseconds ဖြစ်နိုင်ပါတယ်။transition duration ရဲ့ default property ကတော့ zero seconds သို့မဟုတ် no duration ပဲဖြစ်ပါတယ်။
    css transition တွေမှာကျွန်တော်တို့ ဘယ်properties တွေကို transition ပြုလုပ်ချင်တယ်ဆိုတာကိုတိတိကျကျသတ်မှတ်ပေးထားနိုင်ပါတယ်။ကျွန်တော်တို့က browser ကိုဘယ်properties တွေကိုပဲ tranasition လုပ်မယ်လို့မပြောခဲ့ဘူးဆိုရင်၎င်းကproperties တွေအကုန်လုံးကို rule တွေအတိုင်း transition ပြုလုပ်သွားမှာဖြစ်ပါတယ်။ကျွန်တော်တို့က transition property ကိုသတ်မှတ်ပေးမထားရင်တောင်မှာ transition တွေကိုတွေသတ်မှတ်ပေးထားပါလိမ့်မယ်။ဘာ့ကြောင့်လဲဆိုတော့ transition property ရဲ့ default value မှာ all ဆိုတာကိုသတ်မှတ်ပေးထားလို့ပဲဖြစ်ပါတယ်။ကျွန်တော်တို့ကအဲ့တာတွေကိုသတိမထားမိသော်လည်းပဲ transition property ကိုအသုံးပြုချင်းကကျွန်တော်တို့ site ရဲ့ performance ကိုသေးငယ်တဲ့အားနည်းချက်အချို့ကိုဖြစ်ပွားစေပါတယ်။browser က transition အတွက်အပိုအလုပ်တွေကိုလုပ်ဆောင်ပါတယ်။ဘာ့ကြောင့်လဲဆိုတော့အဲ့တာကလိုအပ်တာတွေထပ်ပိုပြီး property တွေကိုပြောင်းလဲဖို့ရန်စစ်ဆေးနေလို့ပဲဖြစ်ပါတယ်။ 

    
    /*===================================
    Button Transitions
    ================================== */
    
    .button {
        color: #fff;
        background: #4a89ca;
        transition-duration: 1s;
    }
    
    .button:hover {
        background: #d36a62;
    }
    
    .button:active {
        background: #a33830;
    }

  • Transition Overlays

    Browser ထဲမှာရှိတဲ့ CSS property တိုင်းမှာ animate မလုပ်နိုင်ပါဘူး။အခုကျွန်တော်တို့ဟာ transitions တေကိုလက်ခံနိုင်တဲ့ properties တွေကိုလေ့လာသွားမှာဖြစ်ပါတယ်။property tables ကိုကျွန်တော်တို့ w3.org မှာလေ့လာနိုင်မှာတယ်။Middle transition state ကို browser ကအတိအလင်းဖော်ပြတဲ့တဲ့အခါမှာ CSS property က animatiable ပြုလုပ်နိုင်ပါတယ်။တခြားနည်းလမ်းနဲ့ပြောမယ်ဆိုရင် property တစ်ခုမှာ point တစ်ခုမှာတစ်ဝက်သတ်သေပြနိုင်တယ်ဆိုရင် အဲ့တာကtransition တွေကို လက်ခံပါလိမ့်မယ်။ဥပမာ display property က animatable property မဟုတ်ပါဘူး။ ဘာ့ကြောင့်လဲ့ဆိုတော့ အဲ့တာတွေက block နဲ့ none သို့မဟုတ် inline စတဲ့ values တွေကြားထဲမှာရှိတဲ့ midway point တွေမဖြစ်နိုင်လို့ပဲဖြစ်ပါတယ်။Lea Verou တည်ဆောက်ထားတဲ့ animatable properties တွေကိုကျွန်တော်တို့ ဒီမှာလေ့လာနိုင်ပါတယ်။ အောက်ပါဥပမာထဲမှာတော့ကျွန်တော်တို့ transition-duration ကို .5s သတ်မှတ်ပြထားပါတယ်။ 

    

    /* ================================= 
    Photo Overlay Transitions
    ==================================== */
    
    .photo-overlay {
        opacity: 0;
        transition-property: opacity;
        transition-duration: .5s;
    }
    
    .photo-overlay:hover {
        opacity: 1;
    }

  • Multi Transitions

    CSS transition တွေက multiple transition တွေသတ်မှတ်ဖို့ရန်ခွင့်ပြုပေးပါတယ်။ကျွန်တော်တို့ကတိကျတဲ့ properties တွေအတွက်မတူညီတဲ့ transitions တွေသတ်မှတ်နိုင်ပါတယ်။အဲ့လိုသတ်မှတ်တဲ့အခါမှာတော့ transition-property ဆိုတဲ့ property မှာနောက်ထပ် value တွေကို comma ခံပြီးထည့်ပေးရမှာပဲဖြစ်ပါတယ်။နောက်ထပ်ကျွန်တော်တို့က transition-duration တွေကိုလည်းခွဲပြီးသတ်မှတ်နိုင်ပါတယ်။မတူညီတဲ့ transition တွေသတ်မှတ်ပြီး duration မှာမထားခဲ့ရင်တော့ရှေ့ဆုံးကတန်ဖိုးကိုသတ်မှတ်ပေးမှာပဲဖြစ်ပါတယ်။ 

    
    /* ================================= 
    Button Multiple Transitions
    ==================================== */
    
    .button1 {
        color: #4a89ca;
        border: 2px solid;
        transition-property: background, border-radius, color;
        transition-duration: .4s, .8s, 4s;
    }
    
    .button1:hover {
        color: #fff;
        background: #4a89ca;
        border-radius: 1em;
    }
    
    button1:active {
        background: #a33830;
    }

    ပြီးခဲ့တဲ့သင်ခန်းစာမှာကျွန်တော်တို့ CSS transition properties တွေကိုလေ့လာခဲ့ပြီးဖြစ်ပါတယ်။ကျွန်တော်တို့က ဘယ် Property တွေကို animate လုပ်မယ်ဆိုတာကိုသတ်မှတ်နိုင်ပြီးတော့ transitions ပြုလုပ်တာဘယ်လောက်ကြာမယ်ဆိုတဲ့ transition duration ကိုလည်းသတ်မှတ်နိုင်ပါတယ်။အခြေခံ transition တွေကိုနားလည်ပြီးဆိုရင်တော့ကျွန်တော်တို့ transition delay ဆိုတဲ့ transition စတဲ့ကြာချိန်တွေကိုသတ်မှတ်နိုင်ပါတယ်။transition delay ဆိုတဲ့ Property ကိုသုံးပြီးတော့ value ထဲမှာတော့ second နဲ့ miliseconds တွေကိုထည့်နိုင်ပါတယ်။transition တွေရဲ့ speed မှာတော့ကျွန်တော်တို့ timing function မှာသတ်မှတ်လိုက်တဲ့အပေါ်မှာမူတည်ပြီးနှေးနိုင်မြန်နိုင်ပါတယ်။အဲ့ဒီ timing functions ကို easing functions လို့လည်းမကြာခနခေါ်ပါတယ်။ ဘာ့ကြောင့်လဲဆိုတော့အဲ့တာတွေကကျွန်တော်တို့ရဲ့ transition လှုပ်ရှားမှုတွေကိုပိုပြီးမြင်နိုင်တဲ့ transition effect တွေဖြစ်အောင်ထုပ်လုပ်ပေးလို့ပဲဖြစ်ပါတယ်။Transition တွေကိုတည်ဆောက်တဲ့အခါမှာအဲ့ဒီ transitionတွေရဲ့နောက်မှာ time interval အစုနေတွပါဝင်နေတဲ့ acceleration curve တွေလိုက်ပါတယ်။အဲ့တာတွေက transition တွေကိုအမြန်နှုန်းမြှင့်နိုင်၊ လျှော့ချနိုင်ပါတယ်။ကျွန်တော်တို့တွေကကြိုတင်သတ်မှတ်နိုင်တဲ့ timing function keyworks တွေကိုအသုံးပြုနိုင်ပါတယ်။အများဆုံးအသုံးပြုတဲ့ keywords တွေကတော့ ease, linear, ease-in, ease-out, နဲ့ ease-in-out တွေပဲဖြစ်ပါတယ်။ဥပမာအနေနဲ့ KEYWORDS အချို့ကိုဝင်ပြီးလေ့လာနိုင်ပါတယ်။အောက်ပါဥပမာထဲမှာတော့ကျွန်တော်တို့ ease-out ဆိုတဲ့ timing function ကိုbuttonမှာသုံးပြထားပါတယ်။ transition properties တွေနဲ့ values တွေကိုအများကြီးရေးနေမယ့်အစားအတိုကောက်တွေရေးပြီးအချိန်တွေလျှော့ချနိုင်ပါတယ်။transition-property နဲ့ transition-durationတွေကိုပေါင်းပြီးရေးနိုင်ပါတယ်။ရေးပုံကတော့ transition ဆိုတာပဲရေးပြီး values တွေကို space ခြားပြီးရေးနိုင်ပါတယ်။Timing function က duration နဲ့ delay ရဲ့ကြားထဲမှာရှိပါတယ်။

    CSS transforms တွေက မတူညီတဲ့ size တွေ၊ position တွေ နဲ့ html elements တွေထွက်ပေါ်လာတဲ့ပုံတွေကိုပြောင်းလဲပြီးပေးနိုင်ပါတယ်။ကျွန်တော်တို့ဟာ transform တွေကိုအသုံးပြုပြီးတော့ content တွေကို two-dimensional သို့မဟုတ် even three-dimensional space တွေမှာ rotating, scaling, skewingတွေကိုပြုလုပ်ဖို့ နဲ့ element တွေကို moving ပြုလုပ်ဖို့လံှု့ဆော်နိုင်တာပဲဖြစ်ပါတယ်။ CSS transformation တွေအကုန်လုံးက single property ဖြစ်တဲ့ transform property မှတစ်ဆင့်ဖြစ်ပွားနိုင်ပါတယ်။အဲ့တာကြောင့်ကျွန်တော်တို့က တစ်ခုသို့မဟုတ်များစွာသော transform functions တွေကိုအသုံးပြုပြီးတော့ကျွန်တော်တို့ရဲ့ page မှာပြောင်းလဲချင်တာတွေကို transform property မှာရေးနိုင်ပါတယ်။transform property အချို့ကို ဒီမှာလေ့လာနိုင်ပါတယ်။rotate ဆိုတဲ့ function ကတော့ 2-d function ဖြစ်ပြီးတော့elements ကို 0 ကနေ 360 degreesလှည့်နိုင်ပါတယ်။ rotate functionရဲ့လက်သည်းကွင်းထဲမှာတော့ angle units အမျိုးအစားတွေအမျိုးမျိုးဖြစ်တဲ့ arguments ကိုလက်ခံတာပဲဖြစ်ပါတယ်။အဲ့တာတွေကတော့ degrees, gradinets, radients တွေနဲ့ turns တွေဖြစ်ပါတယ်။ဒါပေမယ့် developer တွေအများဆုံးအသုံးပြုတာကတော့ degree unit ပဲဖြစ်ပါတယ်။အဲ့တာက circular motion အတိုင်း elements တွေကိုရွေ့ရှားပေးတာဖြစ်ပါတယ်။rotation သတ်မှတ်တဲ့အခါမှာအခြား elements တွေပေါ်မှာသတ်ရောက်မှုရှိမှာမဟုတ်ပါဘူး။

    transform property မှာကျွန်တော်တို့လိုအပ်သလောက် function တွေအများကြီးကိုလက်ခံနိုင်ပါတယ်။အဲ့တာကြောင့်ကျွန်တော်တို့လိုအပ်တာတွေကိုပေါင်းထည့်နိုင်ပါတယ်။ ကျွန်တော်တို့ကြိုက်နှစ်သတ်တဲ့ transform value နေရာတွေမှာထပ်ပြီးထည့်နိုင်ပါတယ်။တကယ်လို့ကျွန်တော်တို့က transform တူတာတွေကိုနှစ်ခါထပ်ရေးမိရင်နောက်ဆုံးရေးခဲ့တဲ့အရာက overwrite ဖြစ်သွားမှာဖြစ်ပါတယ်။ဥပမာထဲမှာတော့ကျွန်တော်တို့ ပုံတွေပေါ်မှာoverflow လုပ်ပြထားတာပဲဖြစ်ပါတယ်။ပုံတွေပေါ်ကိုmouse တင်လိုက်တဲ့အခါမှာဖြစ်ပေါ်လာမှာဖြစ်ပြီး hover မလုပ်ခင်မှာတော့ကျွန်တော်တို့က header တွေ paragraph တွေ button တွေပါဝင်တဲ့ overlay ကိုhidden ပြုလုပ်ထားရမှာဖြစ်ပါတယ်။hover လုပ်လိုက်တဲ့အခါမှာပေါ်လာဖို့အတွက်ကျွန်တော်တို့ကtransform မှာ translateX တွေ Y တွေသတ်မှတ်နိုင်ပါတယ်။ကျွန်တော်တို့ဥပမာရဲ့ဒုတိယ row မှာရှိတဲ့ပုံတွေကိုတော့ slide out ပုံစံမျိုးဖြစ်သွားအောင်ရေးပြထားတာပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ CSS transforms တွေနဲ့ timing functions တွေအကြောင်းကိုလေ့လာခဲပြီးဖြစ်ပါတယ်။ကျွန်တော်တို့ timing function တွေကိုတည်ဆောက်တဲ့အခါမှာ cubic-beziber functions တွေကိုအသုံးပြုပြီးတည်ဆောက်နိုင်ပါတယ်။cubic-beziber functions တွေမှာတော့သတ်မှတ်ပြီးသား function keys အချို့ကိုအသုံးပြုနိုင်ပါတယ်။နောက်ထပ်ကျွန်တော်တို့ timing function တွေကိုဖန်တီးချင်တဲ့ဆိုရင်တော့ cubic-bezier curveမှာကျွန်တော်တို့ကြိုက်သလိုဖန်တီးပြီးကျွန်တော်တို့အသုံးပြုနိုင်ပါတယ်။bezier-curve မှာကျွန်တော်တို့စိတ်ကြိုက် point တွေကိုထိန်းချုပ်နိုင်ပါတယ်။bezier-curve မှာတန်ဖိုး၄မျိုးရှိပြီးတော့အဲ့တာတွေကအစ point နဲ့အဆုံး point တွေကိုသတ်မှတ်ပေးထားတာပဲဖြစ်ပါတယ်။Transition တွေ transforms တွေကိုကျွန်တော်တို့ site တွေရဲ့ visual design တွေမှာအသုံးပြုသင့်ပါတယ်။အဲ့တာတွေကိုကျွန်တော်တို့ကလွယ်ကူစွာအသုံးပြုနိုင်ပြီးအဆင်ပြေစေပါတယ်။

  • Adding 3D Effects

    ကျွန်တော်တို့ဒီတစ်ခါမှာတော့ 3D transforms တွေအကြောင်းကိုလေ့လာသွားမှာဖြစ်ပါတယ်။3D ဆိုတာကတော့ကျွန်တော်တို့လက်တွေ့ဖြစ်ပျက်နေသလိုမျိုးမြင်တွေ့နိုင်တာမျိုးတွေဖြစ်ပါတယ်။ဥပမာ 3D movies တွေမှာဆိုရင်အမှန်တကယ်ဖြစ်နေသလိုမြင်ရမှာဖြစ်ပါတယ်။အဲ့တာတွေက two dimensional movies screen ပေါ်မှာthird dimension effects တွေကိုထပ်ထည့်ထားတာပဲဖြစ်ပါတယ်။Computer ,tablet တွေ mobile screen တွေက two dimensional တွေဖြစ်နေပေမယ့်လည်းကျွန်တော်တို့က CSS transform တွေကိုအသုံးပြုပြီး 3d transforms တွေအဖြစ်ပြောင်းလဲနိုင်ပါတယ်။ပြီးခဲ့တဲ့သင်ခန်းစာတွေမှာတော့ကျွန်တော်တို့ အထက်အောက်ဘယ်ညာရွှေ့ဖို့အတွက် X-axis တွေ Y-axis တွေပေါ်မှာပဲအသုံးပြုပုံတွေကိုလေ့လာခဲ့တာပဲဖြစ်ပါတယ်။အခုတစ်ခါမှာတော့ကျွန်တော်တို့နောက်ထက် Z-axis ကိုပါထပ်ထည့်ပြီးလေ့လာရမှာဖြစ်ပါတယ်။Z-axis က two dimensional screen တွေရဲ့depth ကိုဖန်တီးနိုင်တာပဲဖြစ်ပါတယ်။ဥပမာ Amazon ဆိုတဲ့ size မှာလည်းစာအုပ်တွေဖော်ပြတာကို 3D transform တွေအသုံးပြုထားတာပဲဖြစ်ပါတယ်။
